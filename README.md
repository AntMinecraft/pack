# Ant Pack

## Unicode Ranges

### Overview

| Range               | Type                        |
|---------------------|-----------------------------|
| `\uE000` - `\uEFFF` | Spacing                     |
| `\uF000` - `\uFFFF` | Unicodes                    |

### Spacing
| Range               | Description         |
|---------------------|---------------------|
| `\uEA00` - `\uEAFF` | Negative Spacing    |
| `\uEB00` - `\uEBFF` | Positive Spacing    |
| `\uEC00` - `\uEFFF` | Reserved for future |

### Unicodes
| Range               | Description         |
|---------------------|---------------------|
| `\uFA00` - `\uFAFF` | ant-core            |
| `\uFB00` - `\uFBFF` | ant-soldiers        |
| `\uFC00` - `\uFFFF` | Reserved for future |
